## Instrucciones:
### Clonar repositorio:
```git clone```

### Levantar el Backend:
1.- ```cd node-g71-02```<br>
2.- ```cd backend```<br>
3.- ```npm install```<br>
4.- ```npm run dev```<br>

### Acceder a la aplicación:
```http://localhost:3000```
